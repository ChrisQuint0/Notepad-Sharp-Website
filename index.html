Why Notepad#?

Notepad# is a focused little editor for when you don’t want to wait for a heavy IDE to boot. Pop it open, pick a language template, type your solution, and run it — all in a few key presses.

  • Multi‑tab workflow – keep several files open with modified‑state badges so you always know what changed.
  • CodeMirror One Dark theme – syntax highlighting that feels like a full IDE, without the RAM bill.
  • Language templates – instant boilerplate for C#, C++, Java, and Python with simple keyboard shortcuts.
  • Built‑in code runner – send code to the Piston API, view input/output side by side, and iterate quickly.
  • Minimal shortcuts – just the essentials: format & save (Ctrl + S), new tab (Ctrl + N), close tab (Ctrl + W), open file (Ctrl + O), open runner (Alt + N).

Network access is only required for remote execution through Piston; editing and file operations work completely offline.

Built with Vite + TypeScript + Tauri, running CodeMirror 6 and the Piston HTTP API behind the scenes.

Notepad# was originally crafted for Christopher Quinto’s potato laptop.

What is a standalone file?

A standalone file is a portable version of Notepad#. You don’t need to install anything — just download the .exe, double‑click it, and you’re ready to code.

What is the installer?

The installer sets up Notepad# like a regular Windows application. Run the .msi, follow the prompts, and Notepad# will be added to your Start menu and Programs list.
